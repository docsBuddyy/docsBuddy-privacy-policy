Privacy Policy

Last updated: Monday, 17 March 2025

1. Introduction

This Privacy Policy explains how our extension collects, uses, and protects user information.

2. Information We Collect

We may collect user authentication data through Supabase Auth and store preferences using Chrome Storage API. No sensitive personal data is collected.

3. How We Use Information

  • Authenticate users via Supabase Auth
  • Store user settings and preferences
  • Generate quiz questions using publicly available documentation

4. Data Sharing

We do not share, sell, or distribute any personal information to third parties.

5. Permissions Justification

  • Storage: To save user preferences and session data.
  • Tabs: To detect active documentation pages for quiz generation.
  • Host Permissions: To fetch publicly available documentation content.

6. Security

We implement security measures to protect user data, but users should exercise caution when sharing personal information online.

7. Changes to This Policy

We may update this policy as needed. Users will be notified of significant changes.
